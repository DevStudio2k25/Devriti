# Animation Assets

## Required Lottie Animation

Please download a mental health related Lottie animation and save it as:
`mental_health.json`

### Recommended Sources:
1. **LottieFiles**: https://lottiefiles.com/
   - Search for: "mental health", "meditation", "heart", "wellness", "calm"
   
2. **Suggested Animations**:
   - Meditation person
   - Heart with pulse
   - Brain with sparkles
   - Peaceful/calm animations
   - Wellness/health icons

### How to Add:
1. Download the JSON file from LottieFiles
2. Rename it to `mental_health.json`
3. Place it in this folder: `assets/animations/`

### Fallback:
If no Lottie file is found, the app will show a heart icon as fallback.
